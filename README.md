# 🏥 MediCare AI - Intelligent Medical Appointment System

MediCare AI is an AI-powered healthcare platform for doctor discovery, appointment booking, patient chat, and health record management. It combines modern AI techniques such as Retrieval-Augmented Generation (RAG), LangGraph workflows, vector search, and knowledge graphs to provide intelligent medical assistance.

## ✨ Features

- 🤖 AI-powered medical assistant
- 📅 Doctor search and appointment booking
- 💬 Real-time patient-doctor chat
- 🏥 Health history and medical records
- 🧠 RAG-based medical knowledge retrieval
- 🔍 Intelligent doctor recommendations
- 🔔 Real-time notifications using WebSockets
- 🎤 Voice support and multilingual translation
- 🛡️ Emergency detection and safety checks
- 👨‍⚕️ Admin dashboard for doctor management

## 🛠️ Tech Stack

### Backend

- FastAPI

### AI / ML

### Frontend

- React
- Vite

### Mobile

- React Native
